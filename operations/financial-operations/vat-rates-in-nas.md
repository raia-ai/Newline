VAT rate in NAS:

Different VAT rates may be set up in NAS and charged against different item types. In 2016 the UK standard VAT rate is 20%. VAT on fuel is 5%.

Setup in NAS –

From within a sale go to the setup menu.

Click the File Menu and select the edit VAT rate settings.

The user can add a description and rate as they require.

Against each item type the top VAT rate in the list will appear as default.

The user may edit this to the rate they desire.

These changes then filter through to the calculations made in the cheque and the invoice program.
