NEWLINE ZCARDS - NOTES

JPEGS for cards

Set sysinfo Keys

The files need to be in NLProgs

DLL

copy zbrprinter.dll and zprgraphics.dll to c:\windows\sysywow64 folder

Need to register  with REGSVR32 ZMOTIFPrinter.dll /codebase /tlb

Need to register with Regasm32 ZMOTIFGraphics.dll /codebase /tlb

Other things

Go into the customer screen, then add ins and tick print card option.

To print a card, Tick box in bottom left, print card, then save and it loads zcards

Set printer to zebra printer and print (will be set from sysinfo setting)
