Website Help

Buyers Registration – Website

Providing the website user is registered and logged into the website there will be a Register with Auctioneer button on the website to enable potential buyers to register online.

When clicked the button will disappear and be replace with a pending approval message. The request is sent to NAS in real-time.

Once approved in NAS by the auctioneer staff. On the website in real-time the user will see.

Buyers Registration – NAS

There will be a Buyer Registration option in NAS that will load this form.

It shows the auction staff online requests for buyer registration approval. The online details entered by the online potential buyer are shown. NAS may suggest possible matches in the NAS database based on email address and phone number.

Click Show suggested customer(s)

In this case the email address has found a match. Click select customer if you wish to match the potential online buyer with the NAS customer. Click Cancel and go back if you want to add a new customer to NAS. Click Create New Customer.

Click Save Changes

You can approve the customer and set a credit limit if you wish.

There are various filter options for the display.

Once approved on the website in real-time the user will see.

Setup an online sale

In presale reports and update lot details to online sale you can access the setup for an online sale. Click the coggs \ gears icon, top right and select add/edit online sales.

Give the sale an online name and description. Select the main category the sale type falls into and select the bid type. Set a start and finish date and click save.

Add Photos to Lots – NAS

Go into the sale in NAS. Select Pre Sale Report menu option.

Select update lot details to on-line sale.

Select the sale you want to work in. This will put a tick in the bottom right hand side and click the continue button. Sales with a circle and gavel icon on them are setup as online sales.

You will see all the lots pre-entered into the sale in NAS.

Filters

Show All lots – does indeed do just that.

Show lots already online shows all the lots already uploaded to the website. They have a green background to distinguish them in show all lots mode.

They look like this in list view.

These lots look like this online in Grid view.


To import pictures into NAS click the Import Images button.

Click Add Folder to Import and browse to the folder where you have saved pictures of the lots. The system will load the pictures into the form for matching with the lots. You DO NOT have to rename the pictures from the camera before doing this.

The lots with green backgrounds are already imported in to NAS. To import a lot into NAS click the picture.

Type the Lot number from the picture into the Lot box and press the enter key.

The focus jumps down to the next picture. Type the lot number and press enter and continue the process.

When ready to import to NAS click the Import Images button. This only links the pictures to the lot in NAS, it does not upload to the web at this point.

Click the F8 back button.

Upload Photos to Website

The lots imported in the previous page will be ticked. You can click the + sign to see the picture(s).

Click upload selected to upload photos and lot details to the website.

The lots are uploaded and visible on the website.

Commission Bids – Website

If the user is logged into the website they can leave a commission bid if they have been approved as a registered buyer by the auctioneer.

User on website clicks on view lot details.

If the bid type (in sale setup) is set to commission bid the logged in and approved buyer has a price box to enter a commission bid. The i icon can be hovered over to show the message Your bid is pending approval

Commission Bids Approval - NAS

Select the commission bids option in NAS.

Select the sale and click continue. The user can see what commission bids have been left on the website in real-time.

Tick the box if you are going to accept the commission bid and click import commission bids.

On the website the user will see the bid has been approved. This does not mean they have won it, but just that the auctioneer is aware of it.

Searchable online catalogue

User can search for items of interest.

The user types in Cotes and presses enter.

Watch List

Users can add lots they are interested in to a watchlist making them easy to find and add bids later.
