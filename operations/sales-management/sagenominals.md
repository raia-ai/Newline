Other Deductions Setup and Sage Nominals Setup

In the Installation Menu select Add/Amend Other Deductions/Sage Codes

[pic]

Press the Enter Key to navigate from box to box. When the user  presses  the
enter key on the Commission box a message box appears as shown below.

[pic]

If you only have one nominal for commission the answer is No.  If  you  have
different commission nominals for different sections the  answer  should  be
yes. The user then sees a screen that will enable  different  nominal  codes
to be set up for different sections.

[pic]

To view Other Deductions or add nominals to them, click the Other  Deduction
button.
To Edit the deduction high light it in  blue  and  press  enter,  or  double
click it.

[pic]

To Add an Other Deduction click the Add button.

[pic]

Click the Save button to Save the changes. On the screen above and the  main
Nominal Code page.
