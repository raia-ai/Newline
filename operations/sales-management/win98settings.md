Win 98 – settings


Right click My Computer > Properties> System Properties > Performance tab >
File System > Floppy Disk. Ensure search for new floppy disk drives each
time your computer starts is unticked.

Right click My Computer > Properties> System Properties > Device Manager >
View devices by connection > Plug N Play BIOS > COM1 >Flow Control > Port
Settings > None

Right click My Computer > Properties> System Properties > Device Manager >
Disk Drives > Generic IDE > Properties > Settings > DMA.
