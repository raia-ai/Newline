Show Prize Awards

From within a sale select the utilities menu.

[pic]

Select option 4

[pic]

The user needs to setup how many classes there are  in  the  show,  and  the
number of prizes per class. Click Apply to save the settings.

[pic]

The appropriate number of class rows and prize place columns  will  be  made
ready for user entry into the prize matrix grid.

[pic]

The user should enter a computer account number in the sponsor column,  this
is used to create an invoice against this account number. This  may  be  the
auction firms own account number, or a class  sponsors  account  number.  As
the user navigates through the grid typing the info and pressing  the  enter
key cell with focus is highlighted with a blue rectangle.

Enter the class group. In this example the  class  groups  as  LH   -  Light
Horse; HOYS â€“ Horse of Year Show; FOAL.

Enter the class no, class name and  value  of  prizes  against  each  class.
Click the Save button to save work.


To allocate prize money click the Prize Allocation Tab,  or  press  the  F12
key to move between the Prize Setup and Prize Allocation tabs without  using
the mouse.

[pic]
 [pic]


Enter the appropriate lot number under  the  appropriate  prize  column  and
press the enter key, this will populate the prize name for you and  set  the
focus to the next prize in the class automatically. The prize columns are  a
light yellow colour. It is important for the user to click save, it  is  not
until this button is clicked that the data is saved.

[pic]

Double click the prize cell to create a new  transaction  for  prize  money.
You must have setup which item type code is called prize money in the sale.

[pic] To cross reference the item type enter the  appropriate  code  in  the
textbox and press the enter key to save it.

[pic]

If you click yes a transaction is created against the winning vendor.

[pic]

Producing Prize Cheques
Go into the normal NAS cheque options and there will be cheques for the
prize money transactions you have made.

Back in the Prize allocation software click the [pic] button. This will
retrieve any cheque numbers for lots that have been paid out. This will
populate the cheque no column.

[pic]

Champions

Each class group has a Champion and Reserve row. Enter the lot for the
Class Champion. You have to enter the Champion prize value manually, it
does not work it out from the prize matrix.

[pic]

Double click the Champion lot number to make a champion prize transaction.
The system will prompt you for a sponsor account code. Either enter the
show / mart account code or the actual class group sponsor account code to
generate an invoice against that sponsor.

[pic]
Printing & Exporting options

Click the File menu in the top left of the screen, and select from the
printing or export options as required.

The print options will print immediately to the default windows printer.

The export options will create a csv file and show the user where it has
been saved.
