                   Saleroom.com Data Download Operation ͟͠͞͡

Download Registered Buyers in a Sale.
This option will allow the user to download all registered buyers from the internet to the sale in the
Newline system. To do this you will need to log on to the saleroom.com with your user details and
go in to the required sale.

Once you have selected the required sale you need to select this highlighted option.




This will bring up the following screen and if you scroll down to the very bottom of the screen you
have the following options.




If you select the All option you will then get this screen.
                   Saleroom.com Data Download Operation ͟͠͞͡




Click on the here and you should see this at the bottom of your screen.




Click on Open and you should see a list of buyers details appear in Notepad. If you then click on the
Save As option in notepad and save this file to a central location on the Server that you can browse
to later on to import the details in to the sale on Newline.

this message appearing.




If you click Yes to this screen you should then end up with a screen with the following details for the
sale selected.

If you then select the Save As option and amend the Save as Type drop down to the below settings

Once both of these two options have been completed you have all the required details for updating
the sale in Newline.
