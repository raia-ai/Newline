Intro –

The marts in Northern Ireland have a requirement to display the number of residencies / owners cattle have been on to the point of sale. Mart movements and unsold returns home do not increment the number of residencies. The Rural NI website provides this information on screen for the marts, but provides no download facility. Newline have used screen scrapping techniques to capture and download this into NAS.

Program –

Browse to the market download file

It will run a little slower than before because we are processing the number of residencies now and we did not do this before.

There is a disclaimer label bottom left of the screen please note and be aware.

There is a number of owners column far right of the grid.

Click update to do that.

In 21 view all lots there is an owners column.

Double click the cell for details.

The records in green count the owners. Marts not included in calculation and neither are unsolds returned home.

Breed and Owners Report

filter by exactly the number of owners, or less than a certain number or more than. Click apply to return matches. In this example there are 4 lots with more than 4 owners.

Invoices

A column in the invoices shows the number of owners. This may also be printed on the invoice paperwork.

Salering

Note this will only work on standard 7*30 LED data display boards or media player TVs. Bespoke boards will require bespoke software alterations.

If the user is running the newer layout of salering they will see the number of owners in the lot grid
