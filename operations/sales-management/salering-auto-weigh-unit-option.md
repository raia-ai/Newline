Avery Auto Weigh Option

There is a new salering screen layout available  for this option as of 31/01/2013 version 7.0.339 of salering. To use it these settings are required. Use New Salering Mode must be ticked. Tech note (amends SYSINFO).

There is also a new star printer option which creates a bigger printout and does not require sharing the printer. It just requires the printer name be saved. Tick the Use New Star Printer Mode tick box after entering the printer name.

When going into the salering program if you do attach to an Avery weigh unit the system is connected if the onscreen message says open. If it is closed check the cables are attached correctly to the laptop.

The user clicks the Start Sale button.

The drover or clerk sends weight from the Avery L336. They do this by pressing F8 to weigh the animal(s) and the enter key (This may vary slightly on different units). Type the Lot No and press the enter key. Enter the No Animals and press the enter key. They should get a transmit complete message on the L336 and the data should be sent from the weigh unit to the laptop and appear in the pink list box on the left of the screen.

The focus is on the Next Weight button. If the user wants to bring this weight into the system they just press enter or click the button.

The info is brought across into the weight, lot and heads text boxes.

The focus moves to the Next Lot button. If the user press enter or clicks Next Lot the info is displayed on the display board or TV.

The lot is taken out of the pink box (lots sent from weigh bridge ready to display) and put in the yellow box (showing lots brought throw the system)

The drover sends the next animal(s) from the weigh bridge.

The focus is back on the Next Weight button and the user can press the enter key to load it up ready to be displayed when the first lot is sold. The user presses the Next Lot button which displays the next lots and lets the user sell the one that has just been sold.

The sold or unsold info goes into a grid in the bottom right of the screen.

The user can double click on this grid to edit lots that have already been through the ring.

If the user clicks the Next Weight button and no weight has been sent from the weigh bridge the user will get the following message.

The user can manually type the weight and lot number if they wish or wait for it to be sent from the weigh bridge.

If something has got out of sequence because a beast went a little crazy the user can double click on the yellow box to bring the lot and weight details back for resale.

Lots available for sale within the section are shown in the white listbox on the bottom left of the screen.

As the lots are displaed through the ring they are removed from this list.

If a lot is unsold enter the price as 0 and press the enter key and then press the enter key again when in the purchaser text box.

The system prompts the user

Arrow left and press enter if the lot is unsold (or click the yes button) or press enter if you made a mistake.

The system may be set to prompt for last bid price (it is a user configurable setting)

This information can be seen in the grid in the bottom right of the screen and can be used to sell the animal latter if the seller decides to do a deal.

The user can change sections by clicking the Change Section label. Or get to the configuration options by clicking this label. If they then go back into the same section the lot history for that section on that saleday will be shown again in the gird in the bottom right of the screen.

Media Players or Canteen Displays

If you want to show lots on a media player or canteen display you must tell the laptop which sale ring number you are in. This is displayed at the top of the screen.

and can be set in the Settings menu.

Each laptop/workstation in each ring will need to be setup with a different number.
