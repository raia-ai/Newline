INTRO:

Traditionally many auction firms used one username and password to access NAS for all their users. NAS has the facility to allocate every user a specific username and password. Newline proactively recommend auction firms setup and use different user names for different staff.

BENEFITS:

Improved security

Allows different access rights for different user access levels

Provides a detailed audit of:

Changes to customer accounts

Production of sellers statements

Production of invoices

Record of buyer transfers

Record of sellers transfers

Record of emails sent from NAS

Can have different email addresses and signatures for different users

Links users to their mobile phones to use the NAS mobile app

HOW DOES IT WORK?

Set up users in NAS –

Users with administration privileges may setup additional users by selecting the system configuration option from the main menu.

Followed by view \ setup users.

Click the add user button to add a new user.

Enter the username and password for the new user. Set the access level you wish them to have.

Access Levels –

Admin

Credit Control

General

Rep

Basic

Admin –

CAN ACCESS –

Everything.

Enables the user to log into NAS and navigate all menus and access setup and configuration screens.

Credit Control –

CANNOT ACCESS –

All options in the system configuration menu.

Does not have sufficient access levels to add or edit users. Cannot change certain configuration options in system configuration.

CAN ACCESS -

The sales ledger and see debtor information.

The bank \ cheque reconciliation information.

General –

CANNOT ACCESS –

The system configuration menu.

The sales ledger menu.

The bank reconciliation menu.

CAN ACCESS –

The History menu.

May access turnover analysis. This should be credit control and higher. New turnover menu with 4 options.

May access item type settings.

Rep –

CANNOT ACCESS –

System configuration

Sales Ledger

Bank Reconciliation

History

CAN ACCESS –

Contact Menu

Customer Search

Sale day information and processes

Basic –

CAN ACCESS –

Customer Search

Sale day information

User Audit:

Customer Account

If there is a query on the details on a customer account within the customer search option an admin user may see what amendments have been made to the account over time. Click the view amendments eye icon.

This shows that a mobile phone number was added to the customer account in August 2013. It shows more recently that a second line of address was added and an email address. If old data is removed it will appear in the previous data column. Providing a full audit of changes and which user made them when. This information may help resolve a query or management issue.

Buyers Transfers

Copy Statement & Copy Invoice

Both options show which user took out these statements.
