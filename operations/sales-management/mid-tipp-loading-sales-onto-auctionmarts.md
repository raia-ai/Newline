               Loading Sales onto Auctionmarts.com
               1. login to the website and open the page (‘HELP’) in the navigation bar. (www.dfam.co.uk)




               2. Click the link ‘Login/Register’
               3. This will open a new page asking you to enter your credentials, these can be obtained by
               Newline ASP if you lose the password.




                                MIDTIPP



               4. Once you have entered your credentials, click the login button.
               5. To load a sale onto Auctionmarts click the button ‘Load Sale(s)’




               6. Next Select the Type of sale (Livestock in this instance)
               7. Select the Sale Date (From the dropdown menu)
               8. Select the Sale Frequency (Weekly, Fortnightly, Monthly etc.)
               9. Select the number of sales to generate, if ten selected on weekly frequency it will generate the
                  next 10 weeks sales for the selected day.
               10. Enter a sale description this is usually a brief description about stock types and start times.
               11. Select the stock types within the sale, this is a tick box where you can select multiple types
               12. Once Finished Click Load Sale

               This will load sales onto the Auctionmarts database
