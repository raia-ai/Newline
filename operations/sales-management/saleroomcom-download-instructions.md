https://toolbox.globalauctionplatform.com

Stage 1 - Downloading Bidders




Select “Bidders” under Auction Details and under Current view select “Winners” if available, if not
select “ALL”.




and under Actions select “Export”




Under File Type select “Bidmaster file Format”
This will then save the file from the internet to your “Downloads” folder to a file format like above.




Stage 2 – Importing downloaded Bidder file into newline.




Select “Auto Input Lot Details” followed by “Import Buyer Name and Address Details”




Then browse to your Download folder and select the file that’s just been downloaded. (You may
need assistance with this, if so please contact Newline)
Before clicking on Update check “INTERNET BIDDER” is selected against Select Group. Note the
Paddle No. which allows you to sell those lots purchased on the internet from the sale sheets.
