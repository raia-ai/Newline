New MC2 Production and Aphis Update.

      Setup.

   Go in to each sale and selection option 26 Set up Menu. Select option 2
   Amend sale settings and then click on the options menu at the top of the
   screen.


   [pic]


   From this list select Aphis Mart No. This will display the following


   [pic]


   Enter your Mart number in this box and click on OK.


   Sale operation.


   The MC2 movement forms are automatically produced when the invoice for
   the buyer is printed or the unsold pass out for the seller is printed.
   The first thing that needs to be done is to set up which printer will be
   printing the MC2 documents. To do this you go in to the invoice program
   and follow these steps.


   Select the options menu at the top of screen, then select Printers
   followed by the MC2 Licence option as below. This will display the list
   of available printers on your system and you should select the MC2
   Licence printer.


   [pic]


   Once this has been selected the program will automatically print out the
   MC2 licence when the invoice is printed.


   Once all MC2 licences have been printed you can then send these movement
   details off to Aphis at the end of the sale. To do this you go in to
   option 11 Create Aphis movement file.


   [pic]


This will then display the following screen with the details of the records
to send and any that have already been sent. The records that have not been
sent before will be listed at the top of the list, and will be ticked ready
to send. These will appear in lot number order. The records that have
already been sent will appear in lot number order under the ones ready to
send and will not be ticked.

[pic]

To create the XML file to update to Aphis you need to click on the Create
XML button at the bottom of the screen. This will run through all the
ticked rows in the grid and create a file to update to Aphis. The file name
and location will be displayed on the screen for you to see. Once this file
has been created you can click on the Aphis on Line button and this will
take you to the Aphis ON-Line screen. The name of the file just created
will be at the top of the screen to help you find this when you get to the
file browser screen.

[pic]

Enter your Portal ID and Password and you will see the following screen.
Click on the Aphis Online option in the list below.

[pic]

You then should get the following screen and click on Sign On to continue.

[pic]
When you have successfully signed on you need to click on the Upload Files
option on the left of the screen and then click on the Confirm Moves OUT of
market.

[pic]

This will display the following screen and you should click on the Browse
button and browse to the folder that has the file you previously created.
Once in the required folder select the xml file to upload and enter your
password again.
[pic]
Once you have clicked on Upload you will see a screen telling you to please
wait as the update is in progress. This might take a few minutes to
complete so please be patient. Once the update has completed you will see
the following screen.

[pic]

If everything has updated successfully then you can just click on the
System Notification XML button and this will create an XML reply file that
needs to be updated back in to your sale. If there are any animals rejected
then please make a note of the rejection reasons and contact the Aphis
helpdesk for guidance on what you need to do to resolve these.

When you click on the System Notification XML button you get the option to
save or open the file. If you can click on Save you then get the Save As
screen appear. If you can then just click on the Save option at this point
but make a note of the file name as you will need this later when you come
to update your market system.

[pic]

Once the file has been saved you click on close and you return back to the
Upload of movements out of market screen. At this point you can just click
on the X in the top right hand corner of the screen and you will return
back to the list of animal movement details.

Next option is to click on the Process Aphis Reply button which will
display a list of files to process and you will need to select the filename
that you have just saved. These files normally start with your market
number entered when logging on to Aphis at the log on point and include the
filename that you updated.

[pic]

Click on the file to update and click on open and the system will process
this file and when completed display the message Update Complete.

When all of these processes have been completed you can click on Exit and
Aphis has been updated with all of your cattle movement details for this
sale.
