Date: 19th March 2009

Memo to: Freda Murphy; Saintfield, Auto Bank Rec Input

1: Log onto online banking & view statement.

[pic]


2: Choose your date range. Note that the format the download will be in is
comma separated variable (CSV). The file will be a .OUT file (this is what
the system is expecting) click on Export now to download the file, see
[pic]

Save the file in the bankfile folder on the Z: Log out of online banking.

3. Open NAS and go into;
    • 6. Cheque Reconciliation
    • 2. Auto Bank File Matching

Select the .out file you have just created and it will import it into the
system.

4. Once you are happy with the download click on update and it will auto
reconcile all the listed cheques, after this it will prompt to print
matched and not found cheques.
