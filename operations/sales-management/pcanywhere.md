PC ANYWHERE 32 V8 / V9


Some users have had difficulty with files transferred via file transfer,
not being properly overwritten.

To overcome this, dial your host computer.
1. Set up file transfer. 4th Icon from the left.
2. Go to Tools
3. Choose options
4. At destination file choose “always overwrite duplicate files”
5. Click OK
6. Proceed as usual

The user will only need to do this once.
