Please run this fix on a webadmin PC.



 Please find attached a copy of LSPFIX. Please extract the zip contents
using the password panda. Once the files are extracted please follow the
below instructions





1>     Execute lspfix.exe

2>     Tick the tick box "I know what I'm doing"

3>     Highlight the file pavlsp.dll

4>     Click the remove button in the middle of the program interface, this
should move it to the empty box.

5>     Reboot

6>     Rerun your program and see if the problem remains
