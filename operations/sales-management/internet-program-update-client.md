In the NAS Main Menu click the icon at the top right of the screen.

[pic]

Internet Program Update.

[pic]

You will see a screen with programs published  by  Newline  to  improve  the
functionality of your existing NAS system, or to fix  bugs  that  have  been
reported to Newline. There will  be  a  brief  description  of  the  program
improvement or bug fix, and  possibly  a  pdf  or  word  file  with  further
details (our info sheet).

[pic]

The user will click Install All to install all of the updates, or  highlight
the one they want and install selected.

The Update History Tab shows any updates that have been installed thus  far.
This has a right  click  menu  that  has  options  to  rollback  updates  if
required at any point.
