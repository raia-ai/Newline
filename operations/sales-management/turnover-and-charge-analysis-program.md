New Turnover and Charge Analysis Program.

This new program will give the user the option to be able to enter a from
and too date and get totals for the selected period and then a YTD figures
for all charges used on the seller and buyer side and also number of heads
and total value by animal section.

This new option is option 10 in the Turnover Analysis Menu. The first
screen looks like this.

[pic]

At this point you can enter in a from and too date, they must be in the
sale calendar year, or tick on the View by Sale or Office option. If this
is ticked you get the following selection screen appear.

[pic]

You can then either select a number of different offices to view the
figures by or you can select a number of different sales. Once you have
selected the required details click continue and you will return back to
the first screen.

You will see displayed at the top the selection made from this office /
sale selection screen and then if you press Continue the computer will
calculate the totals and display them on the screen.

[pic]

You can then print these details out by click on the Print button. You will
get the option to select the required printer and the report should then
print.
