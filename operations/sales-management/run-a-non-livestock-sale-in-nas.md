Set up a sale

Click number 10. Auction System from the main NAS menu.

Click setup new sale.

Select the sale you wish to set up.

Enter the sale date

Enter a sale description (it will appear on the invoices)

Click the F5 button to save.

A confirmation message is shown. If the date is correct click ok.

If you are running a non-livestock sale you will be asked.

If you click yes it means as soon as you book a lot into the system it has a status of unsold. You do not have to add a zero price against this lot later to make it unsold. However you do not know if it has been put forward for sale until it is sold.

If you click no the lot status will be pre-booked (not offered yet). It is only unsold or sold when the user enters a zero price for unsold or a price for sold.

How do I access a sale?

From the main menu select option 10. Auction System.

Click 1. Access current sale.

Click the sale and press enter or double click the sale.

The user is then presented with the sale day menu. This is divided into groups to help the user find what they are looking for.

Lot work is done under the lot details group.

Buyer jobs under the buyer details group and so on.

Read the screen. It has lots of useful and helpful labels and information on it.

Produce Item Number Labels

Some non-livestock auctions use item numbers to record which items have been left at the auction site for any given seller. These normally entail a unique number (sticker) applied to the item physical and recorded on a sheet kept by the auction staff. NAS can produce the stickers \ labels for you.

From within the sale day menu select option 2 pre-sale reports.

Select option 2 print lot numbers.

Click the lot range button

Enter the range of item numbers you require and press ok.

Types of Avery label.

To select the type of label click the options menu and configure labels. Select A4 Avery labels.

The drop down combo shows the types compatible with NAS.

The user may configure the layout of the label to include certain fields.

Tick the fields you wish to print out on the label. For printing item numbers this way before the sale only the lot no field needs to be ticked.

Enter items into the sale

From the sale menu select option 1. Pre sale lot entry.

Although you are entering lots you may use the item number and relot later.

Type the seller name

Find and select the seller from the matching list.

Each auction will sell different things and the system may be configured to have item types for sale specific to its needs. Example below.

The user select an item type (by typing in the number and pressing enter.

And enters an item number and presses enter.

The user may wish to enter a reserve price at this point or may not. The system may be configured to automatically stop in this box or bypass it depending on user needs.

The user enters a lot description

Note if the next lot has the same description the user has a quick method of re-entering this by pressing the F7 function key.

If in upper case spell check will not work unless you untick  ignore words in UPPERCASE from the spell check options. Lower case entries will be spell checked.

Click F5 button to save the lot. Note if you do not click F5 or press the F5 function button on the key board it will not save the lot.

The item number will auto increment. Select the appropriate item type (these may be amended later if you are not sure at this point).

When you need to change the seller click the F1 Change button

Re-number items

There is a clear warning to try and dissuade curious auction staff from proceeding unless they are supposed to.

Click yes to continue. Then click the continue button.

The user has another warning. To be sure. It is not the sort of option you want run during a sale in error.

Click yes if you wish to continue.

The re-numbering could be done in the order the items were entered into the sale or alternative you may wish to sort them another way. To sort them another way enter an S and click ok. Leave it as E and click ok if you wish to sort as entered.

Sort a specific way.

The system can sort items by item type, by group or as entered.

Select and click sort.

Enter the lot range for the item types

You can leave spaces between lot ranges if you wish.

Click next

The user is presented with a list of lots and items.

If the user holds their mouse over the grid it shows how to operate this screen.

If I want the antique tub chair to be the first lot offered for sale in the furniture part of the sale the user needs to left click the antique tub chair row. It will go blue

Right click the row you want the selected row to move above.

Click ok if you are happy to move the lot. A new lot order is shown and the lot is moved in position in the grid.

When you are happy with all the changes click the update lot no button and the save button.

The user is prompted to confirm the action.

Sale totals / lot lists

The reports group has useful sale day reports.

Option 21 Sale totals / lot lists shows a list of all the lot numbers and details about them.

Right click the grid for menu options.

Click view all lots.

This report shows the seller, lot number (that was recently allocated in re-numbering process and original item number.

The user can edit and add reserves here if they want to. Click F2 function key to put in edit mode.

Editable fields now have a pink background.

Enter the reserve and press the enter key. The adjustment is saved automatically at this point.

Produce a catalogue

Tick the fields you want in the catalogue.

Select the format and click continue.

The user must have word installed to produce a word catalogue.

The user is asked is this the final catalogue run

If it is click yes. This will then record any edits made after this point and allows a report to be produced of adjustments made to the catalogue which can be used as an appendix to the catalogue.

Basic catalogue layout.

Register buyer numbers

Some auctions will have a list of permanent buyer numbers. These are displayed in the list of registered buyers. Temporary numbers may then be added to different sales using buyer registration.

To register a new buyer select option 15. Buyer registration and enter the surname of the customer you are searching for. You may also enter the first name if you want to restrict the search criteria.

If the customer is not found you may add a new customer

Click ok and Press F1 to add a new customer.

The add customer form opens and the focus is in the buyer number text box. Enter the number and press the enter key.

Enter the buyer name.

Press enterin the address line box if you are using the postcode software finder.

The focus moves down to the postcode. Enter the postcode and press the enter key.

The address will auto populate to street level. You may have to enter the house number or name. Add other details like VAT number, phone numbers and email addresses at this point. Click F5 to save.

Print sale sheets

Select the section and click the continue button

The sale sheet layout appears on screen. Click print.

Sell lots

Enter the lot number and press the enter key.

Enter the price and the buyer number.

Produce Invoices

Press enter to see all buyers on the day.

Alternatively enter:

The buyer number

The computer account number

Their surname

The first letter of their surname

All these methods bring back buyers that match the criteria. Double click from the list to selec the one you require. If there is only one match there is no list it just goes straight into the invoice.

The name and description is shown top right.

Action options bottom left.

Financials far right.

I for invoice

Type I to produce an invoice and press the enter key.

The user is then prompted to pay or not pay.

Select the option you require.

The user sees todays debt (and any historical debt).

Hit return \ press the enter key to take full payment.

Enter the number to select the payment method and press the enter key.

Press enter to confirm full amount by this method. Or if part paying by different methods enter the value for this method.

Check unpaid lots

From the sale menu go into option 21.

Click the sales section, then right click and click view unpaid lots.

This shows a list of all lots that have not been paid for by the buyer. It shows the lot number, seller, lot description and buyer.

Pay out sellers cash

Select option 10 from the sale day menu.

Enter the seller name or computer number

Type A for All Cash payment option. Produce a statement and give the seller the cash.

Move unsold lots to next sale

Open the next sale and go into the lot store option from within that sale menu.

Select option 3 move unsold lots from other sale.

Select the sale you wish to move unsold lots from.

Tick the lots you wish to move to the sale and click the move button.

Close a sale

When all lots have been paid for and the sellers have been paid out and the sale is balanced you should close the sale. This writes any outstanding debtors to the sales ledger and creditors amounts to the ledger as a credit. It also consigns the sale to the history menu.
