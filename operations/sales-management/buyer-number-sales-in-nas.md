                         Buyer Number Sales in NAS.

Buyer Number Sale Setup:

The sale must be setup as a Buyer number sale. To do/check this (from the
main menu);

    • Option 9 System Configuration
    • Option 3 View/setup office details
    • Select the relevant sale on the left of the screen
    • The buyer number system must be set to Y

Permanent Buyer Numbers:

Permanent buyer numbers can be allocated to clients from the main menu. To
do/check this;

    • Option 1 Customer Search
    • Enter, change or delete the relevant permanent buyer number, F5 to
      save

Permanent buyer numbers can also be allocated to clients from the sale day
menu. To do/check this (from the sale day menu);

    • Option 15 Buyer Registration
    • Enter, change or delete the relevant buyer number and tick the
      permanent number tick box.


You can view permanent registered buyers from the main menu in;
    • Option 2 Customer Lists
    • Select Buyer Paddle Numbers & click continue


PLEASE NOTE: Changes and deleted buyer numbers will not be updated in this
list until you have remade the customer name search. To do this (from the
main menu);

    • Option 4 Sales Ledger
    • Option 13 Customer Name Search
    • Remake by either initial or farm address (depending on how you search
      for customers)

Once this has been done any changed or deleted permanent buyer numbers will
appear or be removed from the list of permanent registered buyers.

Temporary Buyer Numbers:

Temporary buyer numbers can also be allocated to clients from the sale day
menu. To do/check this (from the sale day menu);

    • Option 15 Buyer Registration
    • Enter, change or delete the relevant buyer number and save with F5

PLEASE NOTE: To allocate the next available temporary buyer number press
return on the buyer number field when it is blank, this will then
automatically allocate the next number, save with F5.

You can view (combined permanent and temporary) registered buyers from the
sale day menu in;

    • Option 16 List of Registered Buyers

PLEASE NOTE: Changes and deleted buyer numbers will not be updated in this
list until you have remade the customer name search FOR THE SPECIFIC SALE.
To do this (from the sale day menu);

    • Option 25 Utilities Menu
    • Option 1 Remake Customer Index
    • Remake by either initial or farm address (depending on how you search
      for customers)

Once this has been done any changed or deleted buyer numbers will appear or
be removed from the List of Registered Buyers.
NOTE: Any permanent buyer numbers that are deleted from the master folder
will be removed from the sale folder once those sales have been closed.
