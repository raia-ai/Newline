               Convert Powerpoint Presentation to Video.
               1. Open Powerpoint and make amendments to your presentation

               2. Click File

               3. Click Save and Send

               4. Under File Types Select Create a Video

               5. Select the video settings (below are the recommended settings)




               6. Click Create Video

               7. Save the file on to P:\Presentation.wmv, you can get here by clicking the Presentation TV in the
                   favourites window (See example Below)




               8. Click Save this will update the file onto the television displays
