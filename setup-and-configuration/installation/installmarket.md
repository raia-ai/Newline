Installation of New Market Site


1. PRINT OUT ANIMAL TYPE INFORMATION
   Main Menu > Installation Menu > 5. Amend Animal Type Settings > M
   [minimums] > Return until > Print Animal Entry Information on Printer.

The most commonly used types are: Bs, Cs, Ds, Fs, Hs, Ls, Os, Ps, Ss, Us

2. FIND OUT HOW MARKET WORKS?
   WHAT ANIMALS DO YOU SELL?
   HOW DO YOU SELL THEM?
   PREBOOKED?
   MULTIPLE LOT NUMBERS?


3. CHANGE TOLLS, COMMISSIONS & BOOKING-IN INFORMATION ACCORDINGLY.

      Pre-booked stores – WT question:
        B = prompt for weight when pre-booking
        A = prompt for weight when selling pre-booked animals


      Sell during sale – does not matter if A or B, as it only relates to
      pre-book.

      Total WT question:
      [A] or [N] if only 1 head at a time
      [N] mainly used for sheep in multiple numbers
      [T] used for pigs


Cows & Calves can only be pre-booked

CALF                   N

BULL CALVE             H
CALVE                  H

Using Vendor Edit & Setup Main Account No in Bills section.
