System configuration

This menu is for administrators and enables them to set up NAS users and allocate system permission rights. To set up office \ sale template variables.

Setup users

The NAS admin user may setup usernames by clicking the add user button.

Enter the details and select the access level. Click save to setup and save the user.

Access Levels –

Admin

Credit Control

General

Rep

Basic

Admin –

CAN ACCESS –

Everything.

Enables the user to log into NAS and navigate all menus and access setup and configuration screens.

Credit Control –

CANNOT ACCESS –

All options in the system configuration menu.

Does not have sufficient access levels to add or edit users. Cannot change certain configuration options in system configuration.

CAN ACCESS -

The sales ledger and see debtor information.

The bank \ cheque reconciliation information.

General –

CANNOT ACCESS –

The system configuration menu.

The sales ledger menu.

The bank reconciliation menu.

CAN ACCESS –

The History menu.

May access turnover analysis. This should be credit control and higher. New turnover menu with 4 options.

May access item type settings.

Rep –

CANNOT ACCESS –

System configuration

Sales Ledger

Bank Reconciliation

History

CAN ACCESS –

Contact Menu

Customer Search

Sale day information and processes

Basic –

CAN ACCESS –

Customer Search

Sale day information

When a menu option is selected and the user does not have sufficient user privileges they will be warned as shown below.

At the top of NAS it shows the current user that is logged into the system. See current user and it shows the access level. See access type.

Setup offices

This is where the sale templates are set up.

These settings are used in as the defaults when setting up sales.

To edit settings the administrator clicks the F2 – edit button.

Setup email

The NAS users may have different email addresses and email address signatures. These may be setup in here.

Setup TV displays

The TV displays are configured in this option. It may require help from the technical support team.
