Customer Search

There is a new postcode search facility available. Users can purchase an annual licence (the license pays for access to a third party database of addresses) from Newline which enables the user to run a postcode finder within NAS when adding new customers.

Lot Entry

When entering pigs the Slap mark for the seller is kept in the pre-booking screen until the seller is changed.

Selling pre-booked lots

Now shows the user which herd no or holding no is against the account to help decide which the correct buyer account is.

Cheques

The user can block cheque production by setting the seller auto contra field to 2.

Invoices

User can now print buyer number credit cards for all the buyers on the sale day. Improved the grid sorting on the list of invoices.

History

Copy AML2 pig licences are now available in history. There is also a beta version of a sale totals and lot list program in history. Fix a bug when searching for sheep eartags and not finding them.

Contact Management

There are new filter and export options in the contact management software in NAS.

Sales Ledger

Users that wish to produce a receipt to acknowledge payment in the Sales Ledger program can now print out a receipt with a company logo.

Altered the Sage export files so they are lowercase and import into the latest version of Sage.

AMS Sales

Standardised the margin scheme letters throughout NAS.

Sale Sheets

There is a new grid view option in the sale sheets for machinery sales.

APHIS Sheep movement changes

If the lot quantity is different to the number of EID reads recorded against the lot the cell will go green. This is sometimes caused if an A or B lot has been created and the EID reads not adjusted accordingly.

Customer Search

If the postcode database finder is installed NAS has a new option in the customer edit screen.

If this is ticked when adding a new user the following happens. Having entered the surname and pressed enter on address line 1 the focus is taken to the postcode field.

Enter the postcode and press enter and the address details are filled in and you are taken to address line 1 to enter the premise details (eg House No.) This is a street level postcode search it is possible to purchase a premises level but it costs a little more.

Type in the building no. and return will take you to the Country text box and carry on as normal.

Selling pre-booked lots

When pressing enter to the buyer and seeing the matching names in the grid the holding number is now shown in UK mode, or herd no in Northern Ireland or Ireland mode to help the user select the correct account if the buyer has more than one account.

Cheques

Stop payments.

In customer edit set auto contra field to 2.

When doing Q for cheque and P to print the user will be told.

Invoices

Credit card style buyer number cards can be produced with the appropriate card printer and setup by Newline.  In the list of buyers names and addresses on the day the user will see a button print buyers cards.

This option can print out a letter for the buyer stipulating the terms and conditions of having a buyers card.

Contact Management

Contacts can be added to the NAS database either in the customer edit screen (to help with general customer procurement contact) or sales ledger see buyers record screen (to help you keep notes when chasing debts).

Contacts can be viewed in the Contact Management menu.

There is a filter option.

There is now a print or export option.
