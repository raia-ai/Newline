Section settings

Each sale has up to 12 sections. These tend to be stock types or groups of similar items sold in distinct parts of the sale day. Here are some examples of sections in different countries and sectors:

Livestock -

Ireland:

Heifers

Bullocks

Breeding

Canada:

Cattle

Sheep

Pigs

misc

UK:

Prime Cattle (going to slaughter)

Store cattle (to be fattened further for slaughter)

Prime Sheep

Store sheep

Ewes

Pigs

Fur & Feather

Misc

Non-Livestock -

Implements:

Yellow iron

Tractors

Misc

Furniture and fineart:

General

Silver

Glass

Pictures

furniture

Each section may contain many item types. Item types are different item descriptions that have a specific commission value associated with the item type.

Example-

The Canada cattle section may have these item types available for lot input and sell within it.

Feeder

Slaughter

Bred

Other

Each itemtype may attract a different auction commission rate.

Item types may be split down further by the grade or breed.

Auction system

From the main menu select the auction system option.

View sale settings

Select view sale settings and the sale you wish to view.

Amend sale settings

Select amend sale settings.

Select the section you are interested in amending \ viewing by clicking it.

See the item types within that section.

There are options to move the item type up or down in the list. This order is then reflected in lot input lists.

To add another item type to a sale section click the add button.

Select the item type from the left-hand list so it is highlighted and click the > button to move it into the section item type list.

Click the save button to confirm the changes.
