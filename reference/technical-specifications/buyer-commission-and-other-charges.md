Buyer commission

Buyer commission settings may be found within each sale. As each sale may or may not have a requirement for these to differ.

Select the auction system option.

Select the view sale settings option and select the sale you wish to view \ amend.

Select the amend charges, deductions and buyer commission.

Within each sale there may be up to 5 different automatic charges applied against specific item types. These extra deductions are administrator user definable. The user may double click on the name to setup the name of the deduction.

Double click the name label will load an alter deduction form.

Change buyer commission

To change the deduction rate on a specific item type double click the row of the type. This will load a form to amend the settings for that item type.

Enter the amount in the amount text box.

NB: On the buyer side enter a negative figure. This will increase the invoice value. In effect deducting money from the buyer.

In the PVB field enter a value from of either [P V or B]:

P for purchaser \ buyer

V for vendor \ seller

B for apply charge to both sides (on the seller statement and buyer invoice).

In the LHP field enter a value of either [L H or B]:

L for charge by lot

H for charge per head

P for charge a percentage

To charge on unsold lots enter a Y

If the charge attracts tax enter a Y in the vatable field.

It is also possible to set up a min and max charge value.

NB: You must click ok and you will be prompted to update all (of a certain letter) group of 1-9 item types.

Click No to just update one specific item type. Click yes if you know all the item type codes in that letter group and wish to set them all the same.

NB: You must click the F5 save & exit button on the next page to apply these changes. If you do not click it you will lose the changes made on the double click drill down form.

There is a warning message to this effect.

NB: It is very important to test the changes you make to your charges and buyer commission rates. Once you have set these enter a few test lots at different values and double check in the buyer invoices program to ensure the calculations for the commission rates and charges are what you envisage. As this is user configurable it is the responsibility of the user to ensure it is set and working as desired.

Although this option is referred to as buyer commission rates settings it is possible to set the option for sellers or both sellers and buyers in this charge method.

Other deductions

For manual charges that may not be easily automated based on the item type being sold users can set up a flexible list of user defined charges which may be applied in the seller statement or buyer invoice process.

Click the other deductions button.

Click add to add more.

Enter the name of the charge and select Y/N to whether it is taxable. A nominal code may be added for accounting cross reference.
