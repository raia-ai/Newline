Turnover settings

The turnover analysis and stock seller / buyer trading analysis may both be grouped by turnover sections. Each sale has 12 sale sections but the turnover may be broken down further in 24 turnover sections if the user requires this.

System configuration

From within the system configuration menu in the main NAS screen you will find the view \ setup turnover analysis option.

Setup \ edit the turnover section name

Click into the cell name and type the name you require. Click the save button.

Item types within the turnover section

This grid shows which item types are grouped within the turnover section. You may remove an item type from a turnover section by ticking the delete tickbox on the right of the item type and pressing the delete button.

Add item types to a turnover section

Click to select the turnover section then tick the select box on the right of the item type. Click the add button to add.

Warning

An item type may only be in one turnover section. The system will warn you if the item type is already allocated to a section.

Save

You must click save to apply your changes.
