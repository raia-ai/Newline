Introduction

This application allows the user to manage the lots of a sale, to see the buyer of lots and control what pen they are stored in after the point of sale.

Main Screen

The following screenshot is what the user sees when the application is first launched. This screenshot is in Desktop mode. If the application is run on a tablet, font sizes are bigger and the application is limited to full screen mode.

Note: These screenshots are populated with test data

These boxes are the sales, if there are more sales than can fit on the screen, a scroll bar will appear on the right hand side. The user can only select one sale at a time.

This button confirms the selection of the sale and progress the user to the select section screen – This button is only active when a sale has been selected.

These are the window controls and the menu button. When in tablet mode, the maximize/restore window button is missing as the application is limited to full screen mode.

The menu has the setting to manually switch between touch screen and desktop mode as well as a font size slider for touch screen mode.

Section selection screen

This screen allows the user to select the sections of the sale. Each section will show its respective name when the client isn’t using test data.

These items are the sections, when they are selected a green tick will show and the box will be inverted – At least one section must be selected in order to continue.

This button will return the user to the sale select screen.

This button will take the user to the lot edit screen.

This header will display information on the currently selected sale

Lot editor screen

This screen shows all the lots of the selected sections and allows the user to view the buyer of the selected lot and all other lots that belong to that buyer.
The user can also edit the pen number for the selected list.

The lot screen – when a lot has been selected, this field is populated

This is the lot number to indicate which lot the user will be selecting

If the user has selected multiple sections, the section name is also displayed here

This button will hide the respective lot from the list, if the user wishes to see the ignored items, they can check the tick box labelled number 4

This check box will show the all ignored lots in the list

After an item has been processed, it will be removed from the list, the user can check this box to see all of these items on the list

This title displays the name of the section, if the user has selected multiple sections, this will tell the user that multiple sections have been selected

This button takes the user back to the section selection screen

Lot selected

Lot information, here the user can see the lot number, who purchased the lot and their address.

Here the user can modify the pen number of the lot and choose to update the database or cancel the changes they have made.

The text of the selected lot will be made bold to make it clearer which lot has been selected.

This table shows all other lots that have been purchased by the buyer of this lot.
