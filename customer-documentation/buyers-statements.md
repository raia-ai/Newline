3.    Once the individual buyer screen has loaded all options are marked  at
the bottom of the screen.

    • For a buyers LIST press L and  return,  for  a  PASSOUT  press  O  and
      return, see image 6.
Image6.
[pic]

This will print out the relevant report.

   4. Invoicing:

      Select I for invoice, see image 6. Depending on  the  customer  select
one of the following options.

    • D - Deductions
    • P -  Payments
    • U - Unpaid
Image 7.
[pic]

      A.    Deductions; this allows the user to add  or  remove  charges  on
the buyers account, enter the relevant charge number and press return.  Then
input a + (if adding to the invoice) or – (if deducting  from  the  invoice)
and the value. (eg if a customer had taken €10 cash, enter  (charge  number)
8 and press return, then would enter +10 and press  return,  finally  return
to save, see image 8.

Image 8.
[pic]

      B.    Payments; allows the customer to pay for today and pay  off  old
debts. If paying for today enter

    • 1 for today and press return, see image 9.
Image 9.
[pic]
    • Return to pay full amount, see image 10.
Image 10.
[pic]
    • Return to accept or enter another reference if paying off  old  debts,
      see image 11.
Image 11.
[pic]
    • Enter payment type as prompted and press return, see image 12.
    • Return for value or enter the value for that payment type,  see  image
      12.
Image 12.
[pic]
    • If giving a discount leave the cents  outstanding  and  then  allocate
      them to payment type no:5 discount, see image 13.
Image 13.
[pic]

      C.    Unpaid. Select U for Unpaid and then  enter  agent  details  (if
necessary) as shown below.

    • Before printing a prompt for passout will appear, it defaults  to  Yes
      to automatically print the passout as the invoice is  produced.  If  a
      passout is not required select no, see image 14.
Image 14.
[pic]
    • Finally enter any agent details (N for no agent, type in 801  to  list
      all agents and select the relevant one from the list)  and  return  to
      print the invoice and passout, see image 15.
