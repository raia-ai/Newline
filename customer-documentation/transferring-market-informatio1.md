Transferring Market Information: Kivells

   1. Double click on the market system icon on the desktop,  choose  option
      ‘Create Market for Transfer’(see figure 1), type in the  name  of  the
      market you require, when the computer has  found  the  correct  market
      folder type ‘C’ to continue. Type go when prompted - this  will  start
      an automated process.

(Figure 1)

[pic]


   2. Choose ‘M’ for modem when  prompted,  this  will  start  an  automated
      process.

   3. Click onto ‘Liskeard dial up’ icon on the desktop, this will then dial
      into the laptop server at Liskeard.


















(Figure 2)



[pic]


   4. Once the computer has finished dialing you will take  control  of  the
      Liskeard laptop server. Click on the icon called ‘file transfer’  (see
      figure 2), on the left hand side of the screen about half way up, this
      will bring up two halve screens, as a general  rule  the  screen  that
      represents your machine will be on the left as you  look  at  it.  You
      will be able to tell, at the top of each halve screen it will have the
      computer name ie kivserver.

   5. The file you want to send to Liskeard is  called  ‘download.zip’  (see
      figure 3) to find it open the ‘C’ drive by double  clicking  it.  Then
      double  click  the  download  folder,  inside  it  you  will  see  the
      download.zip file (see figure 3), click it once  and  it  will  become
      highlighted. (you may  find  that  when  you  go  into  pcanywhere  it
      automatically goes to this  file  as  we  sent  it  down  to  Liskeard
      previously, if not go through the above instructions).









(Figure 3)

[pic]


   6. Send the download.zip file  it  to  the  Liskeard  machine  (into  the
      download folder on the ‘C’ drive) by doing the following. On the right
      hand side screen (check if this is the  laptop  server  Liskeard,  the
      computer name should give you an indication)  open  the  ‘C  drive  by
      double clicking it, open the download folder by double clicking it.

   7. Then click on file transfer, between the two halve screens toward  the
      top of the window you will see two arrows, one  in  either  direction,
      they file transfer in either direction. (See  figure  3,  there  is  a
      transfer note if you hold the cursor over the icon) You will  need  to
      select the arrow pointing to the right, once you have selected it, you
      will  be  prompted  by  a  window  asking  if  you  want  to  transfer
      information from your machine to the machine  in  Liskeard.  (This  is
      useful as it gives you the opportunity to go  back  if  you  have  not
      selected  the  correct  file).  When  you  are  happy  that  you   are
      transferring download.zip from the kivserver to  the  download  folder
      for Lisekard laptop server click ‘ok’.

   8. Once this has finished you have successfully moved all relevant  files
      to setup the Liskeard sale. Click on ‘end remote session’ (on the left
      hand side of the screen about half way up).





Setting Up the Liskeard Sale.

   1. From the desktop of the laptop  server  double  click  on  the  market
      system icon, choose ‘Receive Central-Market Data’ option  (see  figure
      4).

(Figure 4)

[pic]


   2. After the sale is finished double click on the market system  icon  on
      the desktop of the laptop server. Choose ‘Create Data  for  Return  to
      Central-Market’ option (see figure 4). Choose ‘M’ for modem.

   3. When you are ready to transfer the information back to Holsworthy plug
      the modem cable into the phone socket in the back room and ensure  the
      other end is plugged into the server laptop (WARNING - there  are  two
      points in the back of the laptop that the cable  can  physically  slot
      into, ensure it is in the correct slot, you can tell as  it  will  fit
      tightly, the incorrect slot is larger and the cable  will  be  loose).
      Let Andrew in Holsworthy know that  you  are  ready  to  transfer  the
      market back to Holsworthy.













(Figure 5)

[pic]


   4. Double click on the icon that is named Holsworthy dial-up. The  laptop
      will then dial onto the Holsworthy server, you can tell if it has been
      successful as you will be able to see and use  the  Holsworthy  server
      desktop.

   5. Click on the icon called ‘file transfer’(see figure 5),  on  the  left
      hand side of the screen about half way up,  this  will  bring  up  two
      halve screens, as a general  rule  the  screen  that  represents  your
      machine will be on the left as you look at it. You  will  be  able  to
      tell, at the top of each halve screen it will have the  computer  name
      ie Laptop Server.

   6. The file you want’ to send to Holsworthy is called  ‘download.zip’(see
      figure 6), to find it open the ‘C’ drive by double clicking  it.  Then
      double  click  the  download  folder,  inside  it  you  will  see  the
      download.zip file (see figure 6), click it once  and  it  will  become
      highlighted. (you may  find  that  when  you  go  into  pcanywhere  it
      automatically goes to this  file  as  we  sent  it  up  to  Holsworthy
      previously, if not go through the above instructions).








(Figure 6)

[pic]


   7. Send the download.zip file it to the Holsworhty  machine  –  into  the
      download folder on the ‘C’ drive by doing the following. On the  right
      hand side screen (check if this is the kivserver,  the  computer  name
      should give you an indication) open the ‘C drive  by  double  clicking
      it, then open the download folder by double clicking it.

   8. Then click on file transfer option,  between  the  two  halve  screens
      toward the top of the window you will see two arrows,  one  in  either
      direction, they file transfer in either direction (See figure 6, there
      is a transfer note if you hold the cursor over  the  icon).  You  will
      need to select the arrow pointing to the right, once you have selected
      it, you will be prompted by a window asking if you  want  to  transfer
      information from your machine to the machine in Holsworhty.  (This  is
      useful as it gives you the opportunity to go  back  if  you  have  not
      selected  the  correct  file).  When  you  are  happy  that  you   are
      transferring download.zip from  the  Laptop  server  to  the  download
      folder to kivserver (Holsworthy) click ‘ok’.

   9. This will then begin an automated process that sends all the  relevant
      information to the kivserver  machine  in  Holsworthy,  once  this  is
      finished click on ‘ok’ and then ‘end remote session’ (on the left hand
      side of the screen about half way up). Let Andrew in  Holsworthy  know
      when you have finished.
