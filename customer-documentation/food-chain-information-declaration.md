Food Standard Agency – Food Chain Information Declaration


Newline have added a Food Chain Information Declaration  for  any  livestock
going to slaughter.


Setup

In the invoices option go to the options menu  and  select  printers.  Click
FCI Printer and set it to the printer you want the declaration to print  out
on. When an invoice is produced the declaration will be printed also.

Installation Settings

In the sale installation/setup menu the sections for which these
declarations are applicable should be set as Prime. This should be the case
anyway for all those clients sending MLC data.
