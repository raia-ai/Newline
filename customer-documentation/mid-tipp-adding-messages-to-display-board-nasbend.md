               Adding Messages to Display Board.
               1. Open NAS Software using the usual credentials

               2. Click Addins

               3. Click ‘Add Message to Newline Display’ This will open the below window




               4. From here you can Add a New Message, Edit an existing message and delete a message

               5. Add New Message/ Edit Message brings up the below window


                                     Add Image to the message, this Optional



                  This is where you type the main body of the message, if the text is longer than the
                  screen the text will scroll through the message, in a continuous loop




                You can set a start/finish date for the message here, if you want the message to display
                indefinitely. This can be left at the default settings.

               You can add as many messages as required and the display will loop through each message.
