On the H&H server desktop there is an icon

[pic]

This will link to Stoneleigh and ask for a username and password

HH

letmein

View New Customer Added to NAS

[pic]

HH

LETMEIN

Login in to the system and click Continue to bring back all the new
customers added to the system.

[pic]

If you want to mark these customers as being updated click Yes. Next time
you enter the program and click continue you will not see these customers.
If you click No they will remain marked as being new customers.

[pic]

If the user enters a date range it will not check the customer status, and
it will bring back all new customers added over a given date range whether
marked as seen or not.

[pic]

A matching list of customers is displayed under the name and address tab.
The user can print or export this to excel by clicking the options file
menu and either print or export.

[pic]

The Mailing Groups tab will show any groups these customers belong to.
