Usernames and Access Rights.

[pic]

Double click on the NAS icon to run the Newline Auction System.

[pic]

The user is asked to enter a username and password. The username may be
different for each user and will have a certain level of user access rights
assigned to it. This will determine what the user can see and do within the
system.

[pic]

[pic]

To view what users are set up click the System Configuration option and
View /Setup Users. If you do not have a sufficiently high level of user
access rights you will not be able to enter the configuration menu. You
will get a message saying:

[pic]

If you do have rights then you will be able to view or add new users and
determine their rights.

[pic]

[pic]

There are 6 levels of user rights. To add a new user double click on the
row new to the *.

[pic]



Fill in the details and click save.

BASIC/REP ACCESS CANNOT:

Enter the  sales ledger menu.
Enter the history menu.
Enter the cheque reconciliation menu
Enter the Share Dividend menu
Enter the System Configuration Menu
Close a Sale
Enter the Setup Menu

CREDIT ACCESS CANNOT:

Enter the  sales ledger menu.
Enter the cheque reconciliation menu
Enter the Share Dividend menu
Enter the System Configuration Menu
Close a Sale
Enter the Setup Menu
(But Can Enter the History)

GENERAL /ADMIN/MASTER:

Can access everything
