SPEED CAN BE ENHANCED BY NOT ENABLING FILE OR PRINT SHARING, THIS MEANS
THAT THE WORKSTATION WILL NOT APPEAR IN NETWORK NEIGHBOURHOOD. AS SUCH IT
WILL NOT BE BROADCASTING TO THE BROWSE MASTER. HENCE REDUCING NETWORK
TRAFFIC. IF THE USER DOES NOT SHARE THE HARD DRIVE ANYWAY THEN THERE IS NO
NEED FOR FILE OR PRINT SHARING.
