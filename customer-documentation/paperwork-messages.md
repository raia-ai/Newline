PAPERWORK MESSAGES: CHEQUES; INVOICES; DECALRATIONS

To add a fixed length message to the invoices, cheques or quality  assurance
declaration select the following menu options.

[pic]

[pic]

[pic]

[pic]

[pic]

[pic]

The message will allow the user to type the  maximum  characters  and  lines
that the appropriate paperwork will be able to  display.  Type  the  message
and press enter or use the arrow keys to move up and down.  Click  the  Save
button or press F5 to save the message.  Providing  the  invoice  or  cheque
template is setup for a message it will then appear on the paperwork.
