Note to use this system the user must be a  member  of  esendex  (A  company
that provides An SMS text message service via email for  which  there  is  a
monthly charge. For details see website www.esendex.com

The user must have a current account  number,  username  and  password  from
esendex for the program to work!!

From the vendor and or purchaser stock analysis program select options  that
you require regarding vendor/purchaser,  market  name  and  stock  type  and
click the Run Analysis button.

At the bottom of the list of matching results  you  will  see  an  SMS  Text
button.

 
Click the SMS Text button to show a list of customers from the results  that
have mobile numbers saved against their computer account.

[pic]

If the user clicks the SMS Text button  and  there  are  no  matching  phone
numbers they will see a message to tell them there are no phone numbers  for
customers in the results list.
 
[pic]
 
If there are matches they will see a form to enter an SMS Text message.  See
form below:
[pic]

When you are happy with the message click the Send SMS  Texts  button.   All
the mobiles in the list will be texted.
