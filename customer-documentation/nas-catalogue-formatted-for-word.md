Users can pre-book lots in NAS and format the long catalogue description in NAS to be imported the lots pre-formatted into word in a smart table template.

Enter the lot details as normal.

The user can press the F12 button to format the description as they wish to see it in word.

The user can amend the size of the font, style of the font.

They can bolden. Italic or underline words or paragraphs by selecting the text and clicking the relevant button. The text can be left, middle or right aligned and can be made to have different colours.

Click the apply and save button to save the lot with the formatting.

To create the catalogue click pre sale printouts.

There are a number of different setting the user had available to them.

For lots with pictures loaded against them these pictures can be aligned in different ways and have a lot number beneath them if desired.

If the lots have been loaded with groups then a header can be applied to each group.

The user ticks the options they want in the catalogue and clicks the contiue button. The catalogue may take up to 5 minutes to prepare but will be formatted in a smart template using tables.
