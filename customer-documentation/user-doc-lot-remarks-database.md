INTRO:

User may add remarks against lots in NAS when entering the lots in the pre booking process or when selling the lots. Users type the remarks and there are settings to repeat remarks. In addition to these original options Newline has added a database of standard remarks that user may set up and use against different sections.

BENEFITS:

Speed of entry

Standard and correct spelling

HOW DOES IT WORK?

The setup process -

Go into the lot entry screen and select the options menu and setup \ amend stored remarks.

Here you can setup the remarks you want available to your users when working in a given section.

Or if you prefer abbreviations to keep the sellers statemewnt and buyers invoice tidier.

The remarks are specified against a specific section or sections. In this example the dairy section.

You may also specify the sale \ office they are stored against. This gives the database flexibility.

The user can see what they have stored against each section.
