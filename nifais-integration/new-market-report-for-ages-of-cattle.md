                    New Market report for Ages of cattle.

First you need to set up three new animal grades in the installation menu.
From the installation menu you need to select option 4 LOTINPUT DETAILS.
From the next menu select option 5 ENTER GRADE DESCRIPTIONS.
From the menu select N for ENTER NEW DESCRIPTION.
At the prompt ENTER NEW DESCRIPTION you should enter the description that
you would like printed on the report E.G. Upto 12 Months.
At the ENTER PRINT DESCRIPTION prompt your MUST enter 12 for the upto 12
months animals. 24 for the 13-24 months and 36 for the over 24 months
group.
This will now create the report with the correct age bands required.

                                Grade Report.
Select Grade Report from the Market Reports Menu.
This will then prompt Print Averages by User Entered Types. You would
normally enter N here so that the computer will prompt you to then select
the required section. If you had similar animal types in different sections
E.G you had store cattle steers in section 1 and section 3 you could select
Y at this prompt and the computer would then prompt you to enter the codes
for STEERS and then the codes for HEIFERS.
Once you have entered N you then get a list of sections. Select the
required section.
Then answer the printer and paper prompts.
You then need to specify if you want the report by pence per  kilo or Â£ per
head.
The enter the number of top prices in each range required.
You are then prompted to print averages by section. If you enter N here you
will get the report by Steers andf then Heifers and then any other
individual animal types enter in this section. If however you entered Y
here then the computer will average all animal types in the section
together.
You then have a list of different reports available and you need to select
report 4.
The computer will then prepare the report and once completed prompt you to
press return to print.

This should produce the report in the required age ranges.
