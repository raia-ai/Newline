                         New Input Program 30/06/06


This program will enable users to pre-book stock. It is accessed from the
during the sale menu, option 6 in the LAMS system and access current sale
menu, option 1 in NAS.
Select the  section\sections  that  you  require.  When  you  have  finished
choosing your section\sections press return. (see figure1)
Select vendor, this can be selected in the following ways.
Entering the computer number, ie J80
Entering the 1st three characters of vendors name, ie JON
Entering the complete vendors name, ie JONES
Entering the 1st three characters of vendors name#initial , ie JON#P
Press return.
Figure1
[pic]

Select the customer you require by one of the following methods;
Double click the correct vendor with the mouse.
Use the cursor keys to navigate to the correct vendor and press return  when
they are highlighted blue.
Enter the correct vendors computer number and press return.
Note: New feature, you now can view the holding number and  any  outstanding
balance from this window.
(See figure2)

[pic]



If you need to change any of the section settings, ie  selling  in  guineas,
tick the appropriate box and then click on continue. If there is nothing  to
change simply press return. (see figure3)

Figure3
[pic]

Cattle Entry:

If the vendor is farm assured and there is only livestock  in  the  sections
that are being pre-booked the vendors account  details  will  go  green  and
state farm assured (see figure4). If there are livestock  and  deadstock  in
the sections and the vendor is farm assured it will not go green  until  the
item type is selected.
If the vendor is not farm assured the account details will remain the same.
You can edit the holding  details,  flock  number,  farm  assurance  number,
movement date and  movement  type  for  the  lots  you  are  pre-booking  by
clicking in the relevant field in the yellow box (see figure4).

Figure4
[pic]

Note: New feature, you can press F3 to  view  all  pre-booked  lots  in  the
section you are in (see figure5).

Figure5
[pic]

Note: New feature, you can now amend customers’ accounts by pressing the  F9
key or by clicking on the  ‘amend  customer’  button.  (The  amend  customer
button is  only  activated  once  an  item  type  has  been  selected)  (see
figure6).

Figure6
[pic]

Select the item from the list on the  left  hand  side  of  the  screen.  To
select an item, type the number (from the list on the right  hand  side)  in
the item type text box and press return.
Type the lot number in the lot number text box.
In the number of heads text box type in the number of heads of press  return
and it will default to one.
Before entering cattle for the first time click on the  ‘Tab  Stops’  button
(see figure4).

Select the field that you want the system to automatically stop at. In  this
scenario the system will automatically stop at  Ear  tag  number  D.O.B  and
Breed (see figure8).
Note: New feature, the breed can be selected either  by  its  number  or  by
name, this needs to be setup in the tab stops window.
(The rest of the information is populated by the  customers  details.  Note:
you can manually click on any of the fields in the grid)

Figure8
[pic]

With the ‘Tab Stops’ setup as required  scan  the  cattle’s  passport,  this
should fill in the ear tag, date of birth and breed.
the system will stop at the TB field, if no information  is  required  press
return. You can enter a parish TB rating of 1-4  or  days  left  of  the  TB
test, ie 23 would denote 23 days remaining of the  test.  Press  return  and
the save button is highlighted (see figure4). If there is  nothing  else  to
amend press return and this saves the lot information.
Note: you can click on any of the fields in the  grid  and  manually  change
any of the settings.
New feature, you can now change the sex  of  the  cattle  being  pre-booked,
this means you could have a batch with both male and female cattle.

If the cattle is over 30 months old the date of birth field  will  turn  red
(see figure4). If the cattle is born before 31/07/1996 the system warns  you
of its age (see figure9).

Select the item from the list on the  left  hand  side  of  the  screen.  To
select an item, type the number (from the list on the right  hand  side)  in
the item type text box and press return.

Fine Art Entry

Select the item from the list on the  left  hand  side  of  the  screen.  To
select an item, type the number (from the list on the right  hand  side)  in
the item type text box and press return.
Type the lot number in the lot number text box.
(see figure 12).
You can amend information for vat number, margin scheme  and  agent  details
for the specific lot you are entering by changing them in the yellow box.

Figure12
[pic]

Select the group from the list on the right hand side and  type  the  number
into the group text box.
You can enter a reserve, this can be for the lot or per item in the lot.
You can enter a 40 character description in the short description field.
Note: New feature, In the catalogue description you can enter as  much  text
as is required, this can print on catalogue, sale sheets, vendor  statements
and purchaser invoices.
Note: New feature, you can also add a condition report for  each  lot,  this
can printed in the catalogue (see figure13).
The system also has the facility for a price guide and  a  tick  box  for  a
picture if there is a picture  to  accompany  the  lot  information  in  the
catalogue.
