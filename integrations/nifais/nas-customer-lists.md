The Customer List Option provides many different list reports. Select
option 2. Customer List.

[pic]

Select the report type you wish to view.

[pic]

[pic]

Click the Continue Button.

[pic]

A report will be generated to match your request.

[pic]

To find a particular row in the report you may scroll up or down or use  the
find facility. Right click on the column that you want to search in.  A  new
find form is loaded. In the example below the herd  column  has  been  right
clicked upon and set the search to this column. Enter  the  herd  number  to
search for and press the Find  button.  This  will  highlight  the  row  you
wanted.

[pic]

You do not need to enter the entire search description as there  is  also  a
find next option on the right click option.
