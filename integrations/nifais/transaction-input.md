Enter the computer number or customer name and press the enter key.  If  you
search by computer number the  input  form  will  load.  You  will  see  the
account balance and the batch number.

[pic]

The date defaults to today’s date. Press the enter key to accept  the  date,
or amend as required.

Select the office number either by typing the number or double clicking  the
row.

[pic]

Select the section either by typing the number or double clicking the row.

[pic]

Enter the invoice value or enter a credit with a minus sign in front  of  it
and press the enter key. Enter up to 20 characters of comments if  you  wish
and press enter.

[pic]


Click the save button or press the F5 key.

[pic]

[pic]

A batch running total will be recorded and displayed.

[pic]

[pic]

When the debt is saved the form is unloaded and the  focus  returns  to  the
customer name or number box. If you need to enter another debt for the  same
customer press return, otherwise enter a new number and press return.

[pic]


If you’ve finished the batch press escape or F8. You will see  a  record  of
the batch information you’ve just entered. The user can  print  the  details
or export them to csv.


[pic]

[pic]

If you send the information to sage you will see a copy of the import file.

[pic]

You will see the file name and location.
[pic]

To view past input records click the options menu  and  input  history.  The
user may either search by date or batch number.
