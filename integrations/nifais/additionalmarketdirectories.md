SETUP ADDITIONAL MARKET DIRECTORIES


MAKE A NEW DIRECTORY WITH AN APPROPRIATE NAME, E.G. MON (FOR A MONDAY
MARKET), STORES (FOR A STORE MARKET), OTM (FOR AN OTM MARKET), ETC.

COPY THE MARKET DIRECTORY INTO THE NEW DIRECTORY.

IN THE NEW DIRECTORY DEL OUT*.*

DEL NEWCUST AND NEWBACK

DEL ANY WEEK NUMBERS  CHANGE THE MARKET NAME TO THE APPROPRIATE NAME.

CREATING ADDITIONAL MARKET DIRECTORIES OR COPYING MARKET DIRECTORIES TO
REMOTE SITES:

COPYING FROM A SERVER TO A REMOTE SITE IN WIN 95 OR 98, NOT 311 :

DEL NEW*.*
DEL ??.???  (This will delete all the 2 character file names.)
DEL ?.???   (This will delete all the 1 character file names.)
