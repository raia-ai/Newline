Sheep EID Tags & FCI (Food Chain Information) in History




Copy Invoice

Search for the buyers invoice in history and double click the invoice
required.



[pic]

You see the invoice details, showing lot number, number of heads and
description.

[pic]

New buttons

[pic]


Copy FCI from History

Cick the FCI button. This will show on screen a copy FCI which can be
printed or emailed to the buyer or seller involved.


Copy EID Tags from History

Click the EID Tags button. This will show on screen a copy EID Tags form
which can be printed or emailed to the buyer or seller involved.


EID Tags for individual lots

Click the drill down button   [pic]

Double click the row with the lot number you wish to view at transaction
level.

[pic]

In the livestock frame if there are EID tags for the lot the EID Tags
button will be visible.



[pic]

Click it to see the list.

[pic] It can be viewed, exported to csv or printed.
