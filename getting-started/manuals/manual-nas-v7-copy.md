General Rules

Saving & Exiting:

F5 – Save
Esc or F8 – Exit

In addition -buttons for save and exit will be shown on all forms.




This icon denotes a calendar, clicking on it allows users to change the date




The date selector will default to todays date.

Customers:
The Newline system has one database of customers; a customer could be a seller,
buyer or both.

Definitions:
A seller is a general term for a vendor or consignor.
A buyer is a general term for a purchaser.
Account Numbers:
Account numbers are made up of one letter and a number. The letter is the first
letter of the surname/trading name and the number is the next available number

8|Page
of that letter, e.g. PC Jones = J80, the 80th customer with a ‘J’ surname/trading
name to be registered.

Buyer Numbers:
Buyer numbers can be permanent, temporary or a combination of the two. If a
buyer number is allocated in customer search on the main menu it will be
permanent. If a buyer number is allocated in a sale it will be temporary by default
but can be saved as a permanent number by clicking the permanent tick box.

Old Account Numbers:
If an account number has been brought across from a previous system the
number can be used to search for the customer by adding# in front of the account
number.

Useful information:



The top of the NAS menu shows the version of the software in use, the current
user name, access type level, IP address and usernmae of the device they are
logged onto.
