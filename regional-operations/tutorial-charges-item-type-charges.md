Charges – Item Type Charges

Item type Charges

These can be set within the sale day menus.

Select the sale template you wish to change from the list of sale templates. Select amend sale charges.

This option shows the list of charge descriptions setup in the system configuration. Click the Item Types Charges tab.

Click the Setup New Item Type Charge button to begin the wizard.

Select the Charge

Select the charge name.

Set the default rate

Enter the rate of the charge.

Charge By

Select the formula that determines how the charge is calculated.

Minimum Charge

If a minimum charge applies add it here.

Maximum Charge

If a maximum charge applies add it here.

Trigger Value

Unsold Value

If an unsold charge applies add it here.

Item Types

Tick the item types this charge is to be made on.

Select Offices \ Sale Templates

Select the office template you are in or apply to more than one office template.

Tick the offices you wish to make changes in.

When to begin changes

On step 10 of 10 click finish and the changes will be applied.
