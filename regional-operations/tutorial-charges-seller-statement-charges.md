Charges – Seller Statement Charges

Add a charge to a seller statement

Select the seller and select Q from the action option. Select D for deductions.

Double click the charge name from the left to add a charge

Edit a charge

Override a charge

Double click the system \ item type charge you want to override. Click the override total button.

You will see the system value. Type the value you want to charge instead.

The message box confirms your entry. Click yes if this is correct.

The charge by is changed to a flat rate for overrides. Click yes to continue.

Delete a charge

Double click the charge you want to delete.

Delete a Manual Charge

Click the delete button.

The user is asked delete charge name? Click yes to do so.

Delete a system \ item type charge

If the charge is system calculated, you cannot delete it but you can set the value to 0. Click the delete key and you are prompted. Click yes to make charge 0.

Remove Overrides

If you have made a system override and want to go back to how you had it before click remove overrides. It will step you back a change at a time.
