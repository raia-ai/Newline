Backpen

Backpen allows yard staff to keep track of which animals that have been sold through the ring, or unsold having been offered in the ring but not sold. It shows which pens the animals are in for collection by the buyer \ trucker if sold or seller if unsold.

Click the backpen icon to open the backpen software. Select the sale and click the select salering button.

Select the ring or rings. The user can set different rings to have different background colours to differentiate easier using the slider control.

Select the section.

Each time a lot is sold in the ring it will appear real-time in the backpen, allowing the backpen user to call out the correct pen location for the animals leaving the ring. To aid the yard staff keep control and track of the animals.

Search

Click the search button.

Select the search type criteria from the drop-down combo.

Filter

To filter click a cell you wish to filter on. The click the filter button.

The results are filtered accordingly.

Click un-filter to un-filter the results.

Print

Show pen Report

Click the show pen report button.

Summarises the pens for easy use by the yard staff.

Click hide pen report to return to main backpen options.

Change Pen

Change pen is disabled until you filter by a specific pen.

Once filtered the change pen button becomes available.

Enter the new pen number and click the ok button. The pen number will be changed throughout the system and reflect on the buyer invoices and loadouts.
