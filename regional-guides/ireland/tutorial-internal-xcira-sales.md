RingManager

Xcira – Client utility.

Setup sales \ events

Used to set up sales (they call events). NAS can now be used to set up sales \ events on Xcira and should be if using integrated selling. This is to ensure we get passed back a unique Xcira sale id.

Upload website catalogue

Used to upload the sales catalogue for display on client website.

Used to upload pre-entered lots to online database

Used to upload pre-entered lots. This can now be done in NAS.

Newline login details

Ringmanger URL -> http://apps.ams2.auctionsolutions.com/ringmanager/

User -> nladmin

Pass -> nladmin

Xcira bidder/viewer screen

Used by bidders or viewer of sale info.

Portal to launch a bidder/viewer screen for VJVtest lane = http://apps.ams2.auctionsolutions.com/auctions/attend_links.php?name=VJV%20Test

There is a link at the bottom below the user/password to launch a viewer. If you want to launch a bidder, then user = operations, pass = operations.

Xcira Clerk

Used to clerk the online sale. Show which lot is for sale, edit lot details if needed. Enter the asking price and the winning bid and buyer.

Portal to launch a clerk/marquee for vjvtest lane = http://apps.ams2.auctionsolutions.com/auctions/attend_links.php?name=vjvtest

Clerk login ->

User = clerktest

Pass = clerk

Xcira Marquee

The TV display that shows if the floor or internet has the bid and what the current bid is and who wins the bid.

Marquee login ->

User = marqueetest

Pass = marquee
