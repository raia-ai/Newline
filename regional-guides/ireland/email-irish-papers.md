To email a report file to CSO, Irish Independent or Farmers Journal

From within the sale select option 22 Sale Reports

Select Create CSO file

A summary of the animal data is displayed.

In the File Menu tick the options you wish to use.

It will prompt you to set up the email address for them but gives the user a default address.

Click ok to save this info.

Click Send file to send  the CSO data or click cancel to not send the CSO data but be prompted to send other files.

Answer the questions yes or no as you require.
