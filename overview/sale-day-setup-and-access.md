Sales

A sale in NAS is normally setup for a given date and enables users to input lots for sale by sellers to be sold to and bought by buyers.

Sale day

The sale day grouping from the main NAS screen shows users the options to select from the NAS main menu if they intend to do a task that involves a sale.

Auction system

Select the auction system menu option.

Access current sale

This option lets the user access or select a sale and proceed to input lots, sell lots and so on.

The user may double click the sale they wish to work in to load the sale day menu.

Setup new sale

This options lets the user setup a sale ready for use.

Click the sale from the list so it is highlighted in blue and press the enter key or double click to select.

Enter the sale date

And press the enter key.

Enter a sale description. This is free text and your choice. Examples may be your company name and the day. Or the stock type for sale. E.g. Company name Feeders.

The rest of the detail is pre-populated by initial installation defaults.

Sale type L means it is a livestock sale. The cheque date and BACs date default to the sale date you entered above, although you can amend if you require.

Ensure the tax rate is correct for your country.

Click the F5 setup & save button if you wish to do so.

Other settings

Auctioneers Margin scheme

AMS is a tax treatment scheme for certain UK sales.

Send MLC sale report

This option prevents the sale being closed until certain reports have been produced and sent.

Rep analysis

Should be set to Y if reps are used to procure items and earn a commission on them.

Bank account letter [A-Z]

Different sales may be set up to post the seller cheque details to different bank accounts for reconciliation purposes after the sale.

Buyer number sale [Y/N]

Set to Y if you intend to use a buyer number system for selling.

Bank account type [C/N]

Market type

UK only at present.

The user will see a long hand date prompt to give them an opportunity to double check they have typed the correct sale date. Click ok if you are happy the date is correct.

The system prompt confirms your choice. Click ok.

The sale toggles from the setup new sale option

To the access current sale option.

View sale settings

This option lets an administrative user access the setup and configuration screens for any sale template.

In this example the auction has 7 sale templates it may choose to use.

Download sale to remote server

These options are used when the auction firm has more than one site or has an off-site laptop system for offsite sales.

This option transfers the sale from the main server to the remote server.

Send sale back to main server

This option transfers the sale from the remote server to the main server.
